# ADRs

Architecture decision records for `hermes`.

No standalone ADR has been shipped yet because the runtime is still only one
read-only CLI slice.

Use this directory once HERMES makes a decision that would be hard to infer
from the README alone, such as:

- a live deployment model
- a second read-only question with a new upstream dependency
- the first write action plus approval boundary
- a real gateway or agent orchestration split
